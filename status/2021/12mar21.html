5. CA-R

Noticed that the population numbers in the county popup were wrong, see

https://github.com/daniel-kelley/ca-r/issues/13

This was because the columns moved around. Because of the way the columns were constructed, it wasn't obvious how to use the columns names as a key. Now column lookup uses a regexp ("two problems" nonwithstanding, see

https://softwareengineering.stackexchange.com/questions/223634/what-is-meant-by-now-you-have-two-problems)

and only look up the needed columns so as to not break on unneeded data.
